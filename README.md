# StatusBot

### Telegram Bot for URL Status Checking

#### A simple Telegram bot built with Node.js and Telegraf that checks the availability of a given URL.
#### The bot sends an HTTP request to the provided link and returns the response status code.

#### Supports:

Checking website availability via /status  < url > \
Basic error handling \
Environment-based configuration using dotenv

Tech stack:

JavaScript (Node.js)
Telegraf
